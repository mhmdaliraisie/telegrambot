# ربات کانفیگ و پروکسی تلگرام

ربات برای دریافت کانفیگ V2Ray و پروکسی از کاربران و انتشار در کانال با قالب مشخص.

## قابلیت‌ها

- چک عضویت در کانال اسپانسر قبل از استفاده
- ارسال کانفیگ V2Ray: دریافت لینک/متن کانفیگ، نام انتشار، اپراتور (ایرانسل، همراه اول، رایتل، سامان تل، نت خانگی)
- ارسال پروکسی تلگرام با همان مراحل
- ارسال خودکار به کانال هدف با قالب: عنوان + متن قابل کپی + اپراتور و اطلاعات ارسال‌کننده

## نصب

```bash
pip install -r requirements.txt
```

## تنظیمات

1. فایل `.env` را از روی `.env.example` کپی کنید:

```bash
copy .env.example .env
```

2. مقادیر را در `.env` پر کنید:

| متغیر | توضیح |
|--------|--------|
| `BOT_TOKEN` | توکن ربات از @BotFather |
| `SPONSOR_CHANNEL_ID` | آیدی عددی کانال اسپانسر (مثل -1001234567890) |
| `TARGET_CHANNEL_ID` | آیدی کانالی که کانفیگ/پروکسی آنجا پست می‌شود |
| `SPONSOR_CHANNEL_USERNAME` | یوزرنیم کانال اسپانسر با @ (مثل @channel) |

### گرفتن آیدی کانال

- ربات را ادمین کانال کنید (حداقل حق ارسال پیام).
- یک پیام از کانال فوروارد کنید به @userinfobot یا @getidsbot تا آیدی عددی را ببینید (مثل -1001234567890).

## اجرا

```bash
python bot.py
```

ربات باید در هر دو کانال (اسپانسر و کانال هدف) عضو/ادمین باشد تا بتواند عضویت را چک کند و در کانال هدف پیام بفرستد.
